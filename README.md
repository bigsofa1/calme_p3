# Simple Website Framework

This is a basic static website framework with HTML, CSS, and JavaScript. It's designed for simplicity with no complex features.

## Files
- `index.html`: The main HTML page
- `style.css`: Basic styling
- `script.js`: Simple JavaScript for interactivity

## How to Use
1. Edit the content in `index.html` to customize your site.
2. Modify `style.css` for styling changes.
3. Update `script.js` for any JavaScript functionality.

## Deploying to GitHub Pages
1. Create a new repository on GitHub.
2. Push these files to the repository.
3. Go to Settings > Pages, select the main branch, and save.
4. Your site will be available at `https://yourusername.github.io/repositoryname/`.

## Customization
- Change the title in `index.html`.
- Update colors and layout in `style.css`.
- Add more functionality in `script.js`.
